Bulk Upload

Bulk Upload Instructions

  • You can upload multiple users at once using a CSV file.
  • The CSV must contain the required headers: id, name, email, role.
  • You can download a sample CSV template using the Download button
  • Newly uploaded users will be added to your existing user list.
  • Duplicate emails or invalid entries will be skipped.
  • Make sure the CSV does not contain empty rows.
  • Ensure all email IDs are properly formatted.
  • Only .csv files are supported for upload.
